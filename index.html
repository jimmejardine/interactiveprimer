Interactive Primer

A collection of “smart web pages” that teach from the age of 5 to 105. Everything is one tree of knowledge: start at the roots and climb as far as your curiosity will carry you.

Start at the root → the single page the whole tree grows from.

Sample concepts

  • Counting — a root concept (arithmetic/counting, level 0)
  • Addition — depends on Counting (arithmetic/addition)
  • The Pythagorean Theorem — a geometry entry point (geometry/pythagorean-theorem, level 4)
  • Properties of sin(x) — interactive charts you can tune, with graph-based quiz options (trigonometry/sine-properties, level 5)

These pages demonstrate the full stack: shared look-and-feel via Web Components, KaTeX typesetting, narrated manim-web animations, a self-attested confidence control, and a randomly generated multiple-choice test — all with no build step.
