Computer Science by Subject

Rather than march year by year, you can follow a single subject all the way up — from its first appearance in the foundation year to the specialist final-year course. Sixteen strands run through an undergraduate computer science degree; pick one and climb it.

The sixteen subjects

  • Programming and Software Engineering
  • Data Structures and Algorithms
  • Theory of Computation
  • Operating Systems and Concurrency
  • Computer Architecture
  • Computer Networks
  • Databases and Data Management
  • Compilers and Programming Languages
  • Functional Programming
  • Artificial Intelligence
  • Computer Graphics
  • Distributed Systems
  • Human-Computer Interaction
  • Cyber Security and Cryptography
  • Machine Learning
  • Deep Learning
