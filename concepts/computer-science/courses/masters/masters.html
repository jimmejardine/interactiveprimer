Master's-Level Computer Science

Beyond the undergraduate degree lie the graduate specialisms — focused, research-adjacent courses that assume a solid grounding in linear algebra, probability and algorithms, and take a single subject to the depth of a taught master's (MSc) module. Each course below is a curated path so you can see the whole shape of the syllabus.

Master's courses

  • Computer Architecture — from transistors and the pipeline to caches, multicore, GPUs and accelerators, the quantitative way.
  • Compiler Design — the full pipeline from scanner and parser through semantic analysis, intermediate code and code generation to SSA, dataflow optimisation, register allocation and LLVM.
  • Programming Languages — the theory of what programs mean: the lambda calculus, operational and denotational semantics, type systems from simply-typed to dependent, polymorphism and inference, and the Curry–Howard correspondence.
  • Quantum Computing — qubits and gates through algorithms, error correction and hardware.
  • Data Science — the maths and statistics of the data pipeline, from cleaning and feature engineering to boosting, A/B tests and time series, with hands-on Python delegated to Kaggle Learn.
  • Complexity Analysis and Correctness — how fast and how right: asymptotics, recurrences, amortised and probabilistic analysis, P/NP and the polynomial hierarchy, NP-completeness, approximation and lower bounds, then Hoare logic, weakest preconditions, termination, type soundness, separation logic and model checking.
  • Networks, the Internet & Distributed Programming — the Internet top-down from HTTP to the link layer, then network programming with sockets and the building of real distributed systems: RPC, messaging, load balancing, CDNs and consensus.
