Multiplication

Multiplication is a fast way to add the same amount many times. If you have a equal groups of b things, the total is:

a \times b = c

So 3 \times 4 means three groups of four — the same as 4 + 4 + 4. Press play to build a grid: each row is one group, and we count a whole row at a time instead of one by one — which is really just skip counting.
