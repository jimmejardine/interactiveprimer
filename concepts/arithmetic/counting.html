Counting is how we answer the question “how many?”. We point at each thing once and say the number names in order: one, two, three, …

Watch: each time a new dot appears, we say the next number. One dot, one number — the count goes up by one every time.

Each time we count one more, the total goes up by one. In symbols, if we have n things and add one more, we have n + 1 things.

This is the very first root of the tree of knowledge — almost everything else grows from being able to count.
